# CUDA

#### Compute Unified Device Architecture

- 엔비디아가 개발한 GPU 병렬 컴퓨팅 플랫폼 및 프로그래밍 모델
- 그래픽 처리뿐만 아니라 데이터 분석, 딥러닝 같은 일반 컴퓨팅 작업에도 활용
- 개발자들이 C, C++, Fortran 등 익숙한 프로그래밍 언어로 쉽게 GPU를 다룰 수 있도록 설계되어 있어 사용자 친화적

⇒ GPU 컴퓨팅의 표준이 되어 복잡한 작업도 빠르고 효율적으로 처리할 수 있게 도와줌

## CUDA의 주요 역할과 특징

### 성능 우위

- NVIDIA GPU에 최적화되어 있어 다른 플랫폼보다 우수한 성능을 제공하며 병렬 처리 능력을 최대한 활용하여 연산 집약적인 작업을 가속화 함

### 생태계 및 지원

- 거의 모든 AI 프레임 워크에서 CUDA를 지원하면 NVIDIA가 제공하는 다양한 최적화된 라이브러리로 생산성 향상 가능

### 사용 편의성

- C, C++, Python 등 널리 사용되는 프로그래밍 언어를 지원하여 접근성이 높고, 컴파일러, 디버거 등 종합적인 개발 도구를 포함한 통합 개발 환경을 제공

### 산업 표준

- AI나 고성능 컴퓨팅 분야에서 사실상의 산업 표준으로 자리ㅏ 잡았고 NVIDIA GPU의 성능 우위로 더욱 강화되고 있음
