# RAM vs VRAM

RAM과 VRAM은 둘 다 **임시로 데이터를 저장하는 메모리**지만, 쓰임새가 다릅니다. RAM은 컴퓨터 전체의 작업 공간이고, VRAM은 그래픽 처리 전용 작업 공간입니다. 

## RAM의 기능과 역할

- 운영체제와 실행 중인 프로그램이 빠르게 쓸 수 있는 임시 저장 공간입니다
- 웹브라우징, 문서 작업, 여러 프로그램 동시 실행 같은 **전반적인 시스템 작업**에 영향을 줍니다.
- 용량이 클수록 멀티태스킹이 더 원활해집니다.

## VRAM의 기능과 역할

- 그래픽카드(GPU)에 들어 있는 전용 메모리입니다.
- 이미지, 텍스처, 3D 모델, 쉐이더 같은 **그래픽 데이터**를 저장하고 GPU가 빠르게 처리하게 돕습니다.
- 게임, 3D 렌더링, 영상 편집처럼 그래픽 부담이 큰 작업에서 중요합니다.

## 차이점 항목별 정리

| 항목 | RAM  | VRAM  |
| --- | --- | --- |
| 주 사용처 | CPU가 처리하는 일반 작업 |  GPU가 처리하는 그래픽 작업 |
| 저장하는 데이터 | 프로그램 데이터, 운영체제, 캐시 | 텍스처, 쉐이더, 3D 모델, 프레임 버퍼 |
|  위치 | 메인보드 | 그래픽카드에 탑재 |
| 중요해지는 상황 | 멀티태스킹, 일반 프로그램 실행 | 게임, 고해상도 그래픽, 영상 편집 |
| 업그레이드 | 슬롯 추가나 교체로 확장 가능 | 보통 GPU 교체가 필요해 직접 확장 어려움 |
